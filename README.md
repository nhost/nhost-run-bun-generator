# nhost-run-bun-generator

A delicious Bun Generator application leveraging Nhost Run and the Bun Runtime. In short, Nhost Run is a service that allows you to run your own container images alongside your Nhost project. You can find more details in the [product page](https://nhost.io/product/run) and the [documentation](https://docs.nhost.io/run).

This repository includes:

1. a very basic Bun application
2. a Dockerfile to build a docker image with the Bun application
3. a configuration file to deploy the Nhost Run service
